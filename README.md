# Customer 360 Data Analysis Project

**Report:** [Customer_360_Reprt](DA-Report-Customer-360-Phuong-Chau.pdf)

**Data Source:**
- The dataset is from three csv file: Customer_Registered.csv, Customer_Transaction.csv, Location.csv
- The dataset is stored in "customer 360.sql"
  
**Tools**
- Excel / Data Modeling – Data preparation
- SQL (MySQL) – Data extraction and transformation
- Power BI – Data visualization and dashboarding

# Overview

This project focuses on building a Customer 360 view by integrating and analyzing transactional, behavioral, and external data. The goal is to provide a comprehensive understanding of customer behavior and generate actionable insights to support business decision-making.

The project includes:

- Data analysis report
- Interactive Power BI dashboard
- Structured dataset for analysis

# Report

The report provides detailed analysis and insights, including:

- Data mdeling and matrix
- Overview of customer
- Analyze customers by RFM
- Analyze customers by customer segments
- Key business recommendations.
